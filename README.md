# 谢霆锋成都演唱会开票搜索技能

## 🎯 技能概述
这是一个自动搜索谢霆锋2026年成都进化演唱会开票信息的技能，每天中午12:00自动执行，并将结果以Markdown表格格式发送到飞书。

### 1. 技能安装
技能已自动安装到OpenClaw系统，无需手动操作。


### 4. 定时任务
技能已配置定时任务，每天中午12:00自动执行，无需用户干预。

## ⚙️ 配置说明

### 搜索配置
- `keyword`: 搜索关键词
- `maxRetries`: 最大重试次数
- `timeout`: 搜索超时时间

### 定时任务配置
- `enabled`: 是否启用定时任务
- `time`: 执行时间（12:00）
- `interval`: 执行间隔（daily）
- `timezone`: 时区（Asia/Shanghai）

### 推送配置
- `channel`: 推送渠道（feishu）
- `target`: 推送目标用户ID
- `mode`: 推送模式（announce）
- `format`: 消息格式（markdown）

## 📊 预期输出
搜索结果将以Markdown表格格式发送，包含以下信息：
